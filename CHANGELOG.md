# Changelog

## [0.3.0] - 2026-07-05

### 新機能

- **スケジュール表示（ガントチャート）**: グループ表示・一覧表示に加えて、タスクの期間をタイムライン上にバー / マイルストンで表示する第3のビューモードを追加
  - 前後1年の横スクロール、前月・翌月・今日へ戻るボタン、日付ヘッダーへの曜日表示
  - バーのホバーでツールチップ、クリックでタスク編集
- **カスタムステータス**: 「未着手 / 進行中 / 保留 / 完了」に加えて、任意の名前・カラーのステータスを作成可能に
  - 作成時に「今日のタスク」への表示可否を設定可能（デフォルトステータスは保護され削除不可）
- **期限切れセクション**: 「今日のタスク」と同様に、期限を過ぎた未完了タスクを一覧最上段に自動集約して表示
  - グループ表示・一覧表示・スケジュール表示すべてに対応、経過時間ラベル（例: 3時間前、5日前）を表示
  - セクション内でのドラッグ＆ドロップによる並び替え（順序はローカルに保存）

### 修正

- スケジュール表示でステータス変更ドロップダウンがタスク行の高さで途切れて選択できない不具合を修正
- ランディングページから「macOS向け」の文言を削除（Windows 版も提供しているため）

## [0.2.0] - 2026-06-29

### 新機能

- **タスク期間**: 期日（due_at）に加えて開始日（start_at）を設定でき、タスク一覧に期間ラベルを表示
- **自動アップデーター**: 新バージョンをアプリ内で検知し、バナーからダウンロード・インストールできるように（Ed25519 署名による検証つき）
- サイドバーの UX を改善（今日のタスクの表示条件見直しなど）

## [0.1.0] - 2026-06-25

### 初回リリース

Simple TODO の最初の公開リリースです。

---

### タスク管理

- タスクの作成・編集・削除
- 4段階ステータス管理: **未着手 / 進行中 / 保留 / 完了**
  - タスク名の先頭にステータスバッジを表示、クリックでその場変更
  - 完了ボタン（チェックマーク）で完了↔前ステータスをワンタップ切り替え
- **優先度**（高 / 中 / 低）の設定
- **期日**（due_at）の設定
- **リマインダー**（remind_at）: 指定日時にネイティブ通知で知らせる
- タスクの**コピー & ペースト**（クリップボードバー経由）

### 今日のタスク

- 期日が今日のタスクを画面上部に自動集約して表示

### グループ

- グループの作成・編集・削除（カラー設定対応）
- タスクをグループに分類、グループ単位でセクション表示
- サイドバーからグループでフィルタリング

### タグ

- タグの作成・削除（カラー設定対応）
- 1 タスクに複数タグを設定可能
- サイドバーからタグでフィルタリング

### 並び替え

- 手動並び替え（ドラッグ & ドロップ）
- 期日の昇順 / 降順
- 優先度の高い順

### クイック追加

- ヘッダーのインライン入力からタスクをすばやく追加
- グループ・期日・優先度を追加フォームなしで設定可能

### テーマ

5 種類のカラーテーマを用意。設定はブラウザの localStorage に保存されます。

| テーマ | 背景 | アクセント |
|--------|------|----------|
| Airy   | ホワイト | インディゴ |
| Bloom  | ホワイト | ローズ |
| Noir   | ディープブラック | ブルー |
| Dusk   | ダークブラウン | アンバー |
| Ash    | ピュアブラック | グレー |

### 通知

- macOS ネイティブ通知（`UNUserNotificationCenter`）
- アプリ起動時に通知権限をリクエスト
- remind_at / due_at に基づくスケジューラーがバックグラウンドで動作

### 動作環境

| プラットフォーム | 要件 |
|---|---|
| macOS | macOS 12 Monterey 以降 |
| Windows | Windows 10 / 11 (64-bit) |

### ダウンロード

| ファイル | 対象 |
|---|---|
| `*_aarch64.dmg` | macOS — Apple Silicon (M1 / M2 / M3) |
| `*_x86_64.dmg`  | macOS — Intel Mac |
| `*_x64-setup.exe` | Windows — インストーラー（推奨） |
| `*_x64_en-US.msi` | Windows — MSI パッケージ |

> **macOS をご利用の方へ**: ダウンロード後に Gatekeeper の警告が表示された場合は、  
> システム設定 → プライバシーとセキュリティ →「このまま開く」を選択してください。
