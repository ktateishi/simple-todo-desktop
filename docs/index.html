主要機能

必要な機能が、すべて揃っている

複雑なセットアップは不要。インストールしてすぐに使える、実用的な機能の数々。

☀️

今日のタスクを最上段に

期限が今日のタスクを自動で抽出し、リストの最上段に表示。やるべきことを見落とさない。セクション内でのドラッグ＆ドロップで作業順序も管理できます。

🎯

4段階のステータス管理

未着手・進行中・保留・完了の4ステータスをバッジで直感的に管理。ワンクリックでステータス変更、完了ボタンで素早く Done にできます。

📁

グループ＆タグで整理

仕事・プライベートなどグループで分類し、複数タグでクロス管理。カラーカスタマイズ対応でひと目で識別できます。

🔔

スマートリマインド

期限・リマインド時刻を設定するとmacOSネイティブ通知を送信。アプリを閉じていても通知を受け取れます。

🔀

柔軟な並び替え＆表示

手動・期限順・優先度順での並び替えに対応。グループ表示と一覧表示を切り替えて、自分に合ったビューで作業できます。

🔄

繰り返しタスク

毎日・毎週・毎月・毎年の繰り返し設定に対応。曜日指定や月末ルールなど細かい設定も可能。完了すると自動で次回分が作成されます。

今日のタスク

「今日やること」を
常に把握する

期限が今日のタスクは、グループに関係なく自動的に最上段の「今日のタスク」セクションにまとめて表示されます。他のグループからドラッグするだけで期限を今日に更新することも可能です。

  • 今日期限のタスクを自動収集・最上段表示
  • セクション内でドラッグ＆ドロップで作業順序を変更
  • 他セクションからドラッグで期限を今日に自動更新
  • グループバッジで元グループが一目でわかる

ステータス管理

タスクの進捗を
バッジで可視化

タスク名の先頭に現在のステータスをカラーバッジで常時表示。クリックするとドロップダウンが開き、4つのステータスから即座に切り替えられます。

  • 未着手 / 進行中 / 保留 / 完了 の4ステータス
  • バッジクリックで全ステータスから直接選択
  • 左の完了ボタンで Done ⇔ 前のステータスを往復
  • サイドバーで複数ステータスを同時にフィルター

すべての機能

細部までこだわった
機能の数々

⚡

クイック追加

ヘッダーの入力バーからすばやくタスクを追加。#タグ名 でタグも同時に設定。

🎤

音声入力

マイクボタンを押して話しかけるだけでタスクを追加。ハンズフリーで素早く入力。

🚩

優先度設定

高・中・低・なしの4段階。優先度に応じてタスク左端に色帯を表示。優先度順ソートにも対応。

📋

タスクのコピー&ペースト

タスクをコピーしてペーストバーから貼り付け。タグ・期限・繰り返し設定も引き継ぎ。

🖱️

ドラッグ＆ドロップ

グループ内でのタスク並び替えに加え、今日セクションへのドロップで期限を自動更新。

🔍

全文検索

サイドバーの検索ボックスからタイトルをリアルタイム検索。キーワードですぐに絞り込み。

🌙

ダーク / ライトモード

macOS のシステム設定に連動。目に優しいダークテーマで長時間の作業も快適。

📅

期限管理

期限日を設定すると今日・明日・〇日後などわかりやすいラベルで表示。期限超過は赤色で警告。

🔔

リマインド通知

指定時刻に macOS ネイティブ通知を送信。アプリを閉じていても確実に通知。

🔄

繰り返しタスク

毎日・毎週（曜日指定）・毎月（日付/月末）・毎年に対応。完了と同時に次回分を自動生成。

📝

タスクメモ

各タスクに自由なメモを追加可能。詳細情報や参考リンクなどを記録しておけます。

🗂️

複数ステータスフィルター

「完了以外を表示」など複数ステータスを同時選択してフィルタリング。

🏷️

カラータグ

タグにカラーを設定して視覚的に分類。複数タグの組み合わせで多軸管理が可能。

🚀

自動起動

macOS ログイン時に自動起動。システムトレイ常駐でいつでもワンクリックで開閉。

⚙️

日本語 macOS メニュー

メニューバーも日本語化済み。macOS ネイティブアプリとして自然に使えます。

🦀

高速・軽量

Tauri + Rust バックエンドにより、Electron 比で大幅に軽量。起動も操作もキビキビ動作。
